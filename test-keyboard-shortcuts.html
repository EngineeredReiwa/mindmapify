⚡ Phase 7.1: 高速ノード操作キーボードショートカット

🎯 実装済み機能

目標: ノード作成時間 3秒→0.5秒（6倍高速化）
  • ✅ N キー: 瞬間ノード作成
  • ✅ D キー: 選択ノード複製
  • 🚧 HJKL/矢印キー: ノード微調整移動（実装中）
  • 🚧 Shift+矢印: グリッド整列移動（実装中）

✅ テスト手順

テスト1: N キー瞬間ノード作成

  1. アプリケーションを開く（http://localhost:5173）
  2. キャンバス上にマウスを移動
  3. N キーを押す
  4. 期待結果: マウス位置にノードが作成され、即座に編集モードになる
✨ 従来のツールバークリック→位置クリック（3秒）が、Nキー一発（0.5秒）に短縮

テスト2: 画面中央でのノード作成

  1. マウスをキャンバス外（サイドバーなど）に移動
  2. N キーを押す
  3. 期待結果: 画面中央にノードが作成される

テスト3: 編集中のN キー無効化

  1. ノードをダブルクリックして編集状態にする
  2. テキスト編集中にN キーを押す
  3. 期待結果: 「n」が入力され、新しいノードは作成されない

テスト4: D キー単一ノード複製

  1. N キーでノードを作成し、「テストノード」と入力
  2. ノードを選択状態にする（クリック）
  3. D キーを押す
  4. 期待結果: 右下にオフセットして同じノードが複製される
🎯 複製されたノードが自動選択され、元のノードは選択解除される

テスト5: 複数ノード複製

  1. 複数のノードをCtrl+クリックで選択
  2. D キーを押す
  3. 期待結果: 選択した全ノードが複製される

テスト6: 接続も含む複製

  1. 2つのノードを作成し、接続線で繋ぐ
  2. Ctrl+A で全選択
  3. D キーを押す
  4. 期待結果: ノードと接続線の両方が複製される
🔗 複製されたノード間の接続も自動的に作成される

テスト7: 選択なしでのD キー無効化

  1. Escape で選択をクリア
  2. D キーを押す
  3. 期待結果: 何も複製されない（安全機能）

テスト8: ヘルプ表示

  1. ? キーを押す
  2. 期待結果: ヘルプダイアログでN/Dキーの説明が表示される

🚀 パフォーマンス比較

従来の方法:

ツールバークリック（1秒）→ 位置選択クリック（1秒）→ 編集開始（1秒）= 3秒

新しい方法:

マウス移動（0.2秒）→ Nキー（0.1秒）→ 自動編集開始（0.2秒）= 0.5秒

複製の場合:

従来: ノード選択→コピー→ペースト→位置調整 = 5秒
新方式: ノード選択→Dキー = 1秒
