├─────────────────────────────────────────────────────────────────────────────┤
│                                                 ┌─────────────────────────┐ │
│                                                 │   📋 Mermaid Code      │ │
│                                                 │ ┌─────────────────────┐ │ │

│                                                 │ └─────────────────────┘ │ │

### 要素詳細

**ヘッダーバー**:
- ロゴ「🧠 Mindmapify」（左）
- Undo ↶、Redo ↷、Clear 🗑️、Help ？（右）
- 高さ: 50px
- 背景: #ffffff、下部に薄いボーダー

**メインキャンバス**:
- 背景: #f8f9fa（薄いグレー）
- 中央にヒントテキスト「クリックしてノードを追加」
- クリック可能エリア: 画面全体

**サイドパネル**:
- 幅: 300px（可変、最小250px）
- 初期状態: 展開済み
- コードエリア: モノスペースフォント
- Copy/Previewボタン

## 4. ノード編集モード

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ 🧠 Mindmapify                                  [↶] [↷] [🗑️] [？]          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                 ┌─────────────────────────┐ │
│                ┌─────────────┐                  │   📋 Mermaid Code      │ │
│                │ ┌─────────┐ │ ◄── テキスト     │ ┌─────────────────────┐ │ │
│                │ │ 編集中.. │ │     入力フィールド │ │ mindmap             │ │ │
│                │ └─────────┘ │     （フォーカス） │ │   root              │ │ │
│                └─────────────┘                  │ │     編集中          │ │ │
│                     │                          │ │       サブ1          │ │ │
│         ┌───────────┴───────────┐               │ │       サブ2          │ │ │
│         │                       │               │ │                     │ │ │
│    ┌─────────┐              ┌─────────┐         │ └─────────────────────┘ │ │
│    │ サブ1   │              │ サブ2   │         │                         │ │
│    └─────────┘              └─────────┘         │ [📋 Copy] [👁 Preview] │ │
│                                                 │ [← 閉じる]              │ │
│                                                 └─────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 編集モード仕様
- **トリガー**: ダブルクリック
- **視覚**: ノード内にテキストフィールド表示
- **操作**: Enter で確定、Escape でキャンセル
- **自動サイズ**: テキスト長に応じてノードサイズ調整

## デザインシステム

### 色彩
- **プライマリ**: #007bff（ブルー）
- **背景**: #f8f9fa（薄いグレー）
- **ノード**: #ffffff（白）
- **テキスト**: #212529（ダークグレー）
- **ボーダー**: #dee2e6（ライトグレー）

### タイポグラフィ
- **見出し**: 16px, bold
- **ノードテキスト**: 14px, normal
- **コード**: 12px, monospace
- **ヒント**: 12px, italic, #6c757d

### スペーシング
- **パディング**: 8px, 16px, 24px
- **マージン**: 8px, 16px, 32px
- **ノード間隔**: 最小60px

これらのワイヤーフレームに基づいて、実装フェーズに進むことができます。
